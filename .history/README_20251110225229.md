# Starbucks Project

## Overview
This project consist of a **web homepage** built to demonstrate the use of **semantic HTML5** structure and **custom CSS**.  
It represents a fictional website dedicated to **Starbucks**, a multinational coffeehouse chain headquartered in Seattle, Washington, founded in 1971.


## Objectives
Practice and expand your knowledge of HTML and CSS by creating some pages with layouts
that employ floated elements


## Languages
- **HTML**: for semantic structure and accessibility.  
- **CSS**: for custom layout design and styling.
